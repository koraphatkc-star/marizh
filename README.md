# ระบบข้อมูลสารนิเทศ พัน.ปจว.

## Overview
ระบบข้อมูลสารนิเทศ พัน.ปจว. เป็นระบบที่พัฒนาขึ้นเพื่อจัดการข้อมูลสารนิเทศภายในหน่วยงานที่เกี่ยวข้อง โดยมีวัตถุประสงค์เพื่อให้การจัดเก็บข้อมูล การเข้าถึงข้อมูล และการปล่อยข้อมูลเป็นไปอย่างมีประสิทธิภาพ

## Features
- **การจัดเก็บข้อมูล**: ระบบสามารถจัดเก็บข้อมูลที่เกี่ยวข้องได้หลากหลายประเภท
- **การค้นหาข้อมูล**: ผู้ใช้สามารถค้นหาข้อมูลได้อย่างรวดเร็ว
- **การรายงานผล**: ระบบสามารถสร้างรายงานต่าง ๆ ที่เกี่ยวข้องกับข้อมูล

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/koraphatkc-star/marizh.git
   cd marizh
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Usage
- เข้าสู่ระบบด้วยข้อมูลผู้ใช้ที่ได้รับ
- เลือกเมนูที่ต้องการใช้งาน
- ทำการจัดการข้อมูลตามต้องการ
